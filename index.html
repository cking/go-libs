The Libraries

Scraping

  • https://github.com/yhat/scrape

    Some simple and helpful additions to x/net/html

  • https://github.com/ericchiang/css

    CSS Selectors for x/net/html

Testing

  • https://onsi.github.io/ginkgo/

    BDD-Style test runner which is compitable with testing from Go

  • https://onsi.github.io/gomega/

    Simple and easy assert library (or matcher, in their terms)

Misc

  • https://github.com/apex/log

    Apex Logging Framework I prefer this one over other solutions, because it supports different formatters per output

  • https://github.com/olebedev/config

    Configuration for your project from several sources (files, environment variables, program arguments, ...)

    Go bindings to Discord
